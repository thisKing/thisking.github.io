一、基本概念

二、工作原理

  • 利用 viewport 和设备像素比调整基准像素

CSS像素、设备像素、逻辑像素、设备像素比

CSS像素、逻辑像素 === CSS px像素单位
设备像素：设备机器手机上的物理像素（不同版手机不一样）
设备像素比：CSS像素和设备(物理)像素的比值：1px像素有2个设备(物理)像素点，1比2。dpr

rem 和 em 像素单位
  • rem // 相对 html 标签的 font-size 单位
  • em // 相对父级 font-size 单位
